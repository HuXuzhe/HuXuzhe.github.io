Python概述

  • 一种面向对象的解释型计算机程序设计语言
  • 具有丰富且强大的内置库和第三方库
  • 语法简洁灵活
  • 开源、跨平台

应用场景：

  • 数据分析
  • 人工智能
  • 网络爬虫
  • 自动化运维

  • 注释的作用

    描述一段代码的实现逻辑和功能，增强代码可读性，易于维护

格式：变量名 = 数值/字符串
